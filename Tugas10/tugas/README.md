# Tugas 7 - News App Welcome Screen

Aplikasi Flutter dengan tampilan welcome screen untuk aplikasi berita.

## Cara Menjalankan

1. Pastikan Flutter sudah terinstall
2. Jalankan `flutter pub get` untuk menginstall dependencies
3. Jalankan `flutter run` untuk menjalankan aplikasi

## Fitur

- Tampilan welcome screen dengan ikon berita custom
- Tombol "Masuk" dan "Mendaftar"
- Design yang clean dan modern
